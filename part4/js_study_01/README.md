- 주어진 값과 일치하는 첫번째 인덱스 출력

## .slice(숫자1, 숫자2)

# 숫자
## .toFixed(숫자)   //  문자데이터로 출력됨
- 소수점(숫자) 만큼 출력

## parseInt() // 전역함수 (숫자 데이터)
- 문자데이터를 분석해서 숫자만 추출해서 정수로 출력

## parseFloat() // 전역함수 (숫자 데이터)
- 문자데이터를 분석해서 숫자만 추출해서 소수점 유지후 출력

## Math.abs()
- 주어진 숫자의 절대값을 반환
