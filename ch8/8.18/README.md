플렉스 (정렬) container(1)

display : Flex Continer의 화면 출력(보여짐) 특성
  flex : 블록 요소와 같이 Flex Container 정의
  inline-flex : 인라인 요소와 같이 Flex Container 정의

flex-direction : 주 축을 설정
  row : 행 축(좌==>우)(기본 값) ....수평
  row-reverse : 행 축(우==>좌) ....수평
  column : 열 축(위=>아래) ....수직
  column-reverse : 열 축(아레=>위) ....수직
