플렉스 (정렬) Items

order : Flex Item의 순서
  0 : 순서 없음(기본 값)
  숫자 : 숫자가 작을 수록 먼저

flex-grow : Flex Item의 증가 너비 비율
  0 : 증가 비율 없음(기본 값)
  숫자 : 증가 비율

flex-shrink : Flex Item의 감소 너비 비율
  1 : Flex Container 너비에 따라 감소 비율 적용(기본 값)
  숫자 : 감소 비율

flex-basis : Flex Item의 공간 배분 전 기본 너비
  auto : 요소의 Content 너비
  단위 : px,em,rem 등 단위로 지정
