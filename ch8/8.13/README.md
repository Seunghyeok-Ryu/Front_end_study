문자

color : 글자의 색상
rgb(0,0,0) : 검정색 (기본 값)
색상 : 기타 지정 가능한 색상

text-algin : 문자의 정렬 방식
left : 왼쪽 정렬 (기본 값)
right 오른쪽 정렬
center : 가운데 정렬
justify : 양쪽 정렬

text-decoration : 문자의 장식(선)
none : 장식 없음 (기본 값)
underline : 밑줄
overline : 윗줄
line-through : 중앙선

text-indent : 문자 첫 줄의 들여쓰기
0 : 들여쓰기 없음
단위 : px,em,rem 등 단위로 지정
음수 사용시 내어쓰기
